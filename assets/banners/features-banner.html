✦ FEATURES ✦

❄️
FREEZE MECHANIC
Taggers left-click runners to freeze them in place. Frozen players cannot move or jump until unfrozen by a teammate.
🧊
⚔️
ROLES & CLASSES
Choose Tagger or Runner preference. Select a class with unique abilities from the hotbar class menu.
🛡️
🗳️
MAP VOTING
Join the Vote Lobby and vote for your favourite map. The map with the most votes wins when the countdown ends.
🗺️
🏆
MULTI-ARENA
Run multiple arenas simultaneously. Set arenas as vote-only so they only appear in the vote lobby.
🏛️
📊
LIVE SCOREBOARD
Dynamic sidebar scoreboard shows timer, frozen count, role, class, and live vote standings in the lobby.
📋
🌐
VOTE LOBBY
A universal pre-game lobby with rainbow map voting item, role selection, and automatic game launch on countdown end.
🎮
